# Content Pillars — Always-On Social Engine

This document defines the **4 evergreen content pillars** for TathastuKeepsakes social media. Each pillar includes:
- Purpose
- Fill-in-the-blank caption template
- 2 fully-written examples using real products
- Hashtag set reference

Campaign files (02–05) and the master calendar (00) reference these pillars by name. Use this as your copy-paste toolkit.

---

## Usage Notes

- **Offers = placeholders.** If a campaign post needs an offer mention, write `{{FESTIVAL_OFFER: e.g. 10% off orders above ₹999}}`. The founder fills or deletes before posting.
- **Verify before posting:** Always double-check product IDs, prices, and links against `lib/products.json` or the live site.
- **WhatsApp:** Manual send only, opt-in audiences, personal tone. See `06-whatsapp-playbook.md` for detailed guidance.
- **Shot references:** Image/video shot IDs (e.g., `SHOT-GANESHA-01`) are defined in `07-image-shotlist.md`.
